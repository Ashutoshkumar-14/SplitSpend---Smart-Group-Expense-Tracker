# 📌 About the Project

**SplitSpend** is a smart group expense tracking web application that helps users split, manage, and settle shared expenses easily. Whether it’s rent, travel costs, food bills, or team expenses, the platform ensures fair and transparent expense management.

The project is inspired by modern fintech applications like Splitwise and enhanced with:
- 🤖 AI-assisted expense insights
- 📊 Expense analytics and reports
- 📷 Receipt scanning using OCR
- 📍 Location-tagged expenses
- 🌐 Responsive modern UI

---

# 🎯 Problem Statement

Managing group expenses manually often creates:
- Confusion in settlements
- Unfair payment distribution
- Lack of transparency
- Difficulty tracking shared spending

**SplitSpend** solves these problems by providing a centralized and automated expense-sharing system.

# 🚀 Features

| Feature | Description |

| 🔐 User Authentication | Secure login/signup using JWT authentication |

| 🤖 AI Suggestions | Simplified debt settlement recommendations |

| 🌍 Location Tagging | Add expense locations for better tracking |

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Ashutoshkumar-14/SplitSpend.git
cd SplitSpend
```

## 2️⃣ Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

# ▶️ Run the Project

## Start Backend

```bash
cd server
npm run dev
```

## Start Frontend

```bash
cd client
npm start
```

---

# 🔑 Environment Variables

Create a `.env` file inside the `server` folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
